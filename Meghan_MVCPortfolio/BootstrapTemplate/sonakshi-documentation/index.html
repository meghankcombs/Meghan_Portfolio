H) FAQ - top

1) This template main color is - #083c5a. if you want to change this color. just replace it with your own color. you can get lots of color code here - material color

2) If you want to change contact mail id, just open "contact.php" and you see the mail address. just change it with your own mail address.

3) For customise the google map location just go to the template folder. Then go to assets -> js , then open "scripts.js" using editor.
Then you can see the comment line /* Contact Map*/ .
Go to this url. give your address and take your own location google map - http://www.gps-coordinates.net and Change map coordinate.
